yo nodejsapp
=========
nodejs应用开发基本环境设置
用 yo webapp 生成的框架基础上改造成 nodejs使用的app， 
同时在后段集成了koa，在前端集成了requirejs.


## Getting Started


### Usage

git it in your work directory

```bash
$  git clone https://github.com/loufq/yo-nodejswebapp.git
```

```bash
$  cd yo-nodejswebapp
```

```bash
$  bower install
```

```bash
$  cd app
```

```bash
$  npm install 
```
[nodemon](https://github.com/remy/nodemon)
```bash
$  nodemon app.js
```

Have Fun
